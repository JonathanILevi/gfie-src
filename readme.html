Uninstalling GFIE properly:

If you used an installer program to set up GFIE, you have to select Control Panel|Programs and Features, browse for Greenfish Icon Editor Pro in the list appearing, and choose Uninstall.

If you unpacked a portable version of GFIE, you have to:

  1. Make sure that no file types are associated with GFIE. Uncheck all previously set associations at Settings|Preferences|Install.
  2. Delete the program folder.
